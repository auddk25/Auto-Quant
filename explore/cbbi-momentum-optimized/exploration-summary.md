## 优化过程

### 第 1 层: 参数扫描 (32 组)

测试了 4 个方向的参数变化：
1. **动量周期**: ENTRY_MOM (2-5d), EXIT_MOM (3-6d)
2. **CBBI 阈值**: 0.50-0.75
3. **出场阈值**: -0.02 到 -0.05
4. **趋势过滤**: EMA50/100, EMA100/200

**关键发现**: EXIT_THRESHOLD 对收益影响最大
- EXIT_THRESHOLD=-0.02: +951.7% (最优)
- EXIT_THRESHOLD=-0.03: +732.7% (原参数)
- EXIT_THRESHOLD=-0.04: +506.2%
- EXIT_THRESHOLD=-0.05: +389.5%

### 第 2 层: 组合优化 (243 组)

测试了 5 个参数的完整组合：
- ENTRY_MOM: [2, 3, 4]
- EXIT_MOM: [3, 4, 5]
- CB_THRESHOLD: [0.55, 0.60, 0.65]
- EXIT_THRESHOLD: [-0.02, -0.03, -0.04]
- EXIT_CBBI: [0.75, 0.80, 0.85]

**最优参数组合**:
```python
ENTRY_MOM = 3          # CBBI 3日动量 (不变)
EXIT_MOM = 3           # CBBI 3日动量 (原为 4)
CB_THRESHOLD = 0.65    # CBBI < 0.65 才入场 (不变)
EXIT_THRESHOLD = -0.02 # 动量 < -0.02 出场 (原为 -0.03)
EXIT_CBBI = 0.80       # CBBI > 0.80 出场 (不变)
```

## 关键洞察

1. **EXIT_MOM=3 优于 EXIT_MOM=4**: 更快出场信号更好，减少回撤
2. **EXIT_THRESHOLD=-0.02 是最优阈值**: 更敏感出场，抓住更多利润
3. **EXIT_CBBI 对收益影响较小**: 0.75-0.85 差异不大
4. **ENTRY_MOM=3 是最优入场周期**: 3 日动量最稳定
5. **参数变化小，逻辑一致**: 过拟合风险低
