# CBBI + AHR999 日线抄底策略设计

> 日期: 2026-04-29
> 分支: strategy-cbbi-ahr999
> 目标: 回撤小 + 收益不输 BuyAndHold，同时探索超越 CbbiMomentum 的可能

---

## 背景

114 轮均值回归迭代已耗尽参数空间。CbbiMomentum (+732.7%) 是当前最优策略，但基于 1h timeframe + CBBI 动量。

本次探索转向 **日线级别**，使用 **CBBI + AHR999 双指标组合**，目标两个方向：
1. 收益 ≥ BuyAndHold (+88.5%)，同时回撤远小于它
2. 绝对收益超越 CbbiMomentum

---

## 策略基础

```python
class CbbiAhr999Daily(IStrategy):
    timeframe = "1d"          # 日线级别
    can_short = False         # 现货
    max_open_trades = 1
    process_only_new_candles = True
    startup_candle_count = 200
    minimal_roi = {"0": 100}  # 无止盈，依赖出场信号
    stoploss = -0.25          # 固定止损 25%
```

**指标加载**：
- CBBI：`merge_cbbi()` — 链上综合情绪指标 (0~1)
- AHR999：`merge_ahr999()` — 定投指标，<1 表示"便宜"

## 入场逻辑变体

### A. 双指标阈值抄底

```
IF CBBI < cb_threshold AND AHR999 < ahr_threshold:
    入场
```

参数空间：
| cb_threshold | ahr_threshold | 含义 |
|:---:|:---:|---|
| 0.30 | 0.40 | 极度恐惧 + 极度便宜 |
| 0.30 | 0.60 | 极度恐惧 + 便宜 |
| 0.30 | 0.80 | 极度恐惧 + 偏便宜 |
| 0.35 | 0.40 | 恐惧 + 极度便宜 |
| 0.35 | 0.60 | 恐惧 + 便宜 |
| 0.35 | 0.80 | 恐惧 + 偏便宜 |
| 0.40 | 0.40 | 偏恐惧 + 极度便宜 |
| 0.40 | 0.60 | 偏恐惧 + 便宜 |
| 0.40 | 0.80 | 偏恐惧 + 偏便宜 |

→ 9 种组合

### B. 动量回升入场

```
IF CBBI_N日动量 > 0 AND AHR999_N日动量 > 0:
    入场
```

参数空间：
| N | 含义 |
|:---:|---|
| 3 | 短期动量（类似 CbbiMomentum） |
| 5 | 中期动量 |
| 7 | 长期动量 |

→ 3 种组合

### C. 绝对值 + 动量混合

```
IF CBBI < cb_threshold AND AHR999 < ahr_threshold AND CBBI_N日动量 > 0:
    入场
```

在 A/B 粗筛结果出来后，取最优参数组合测试。

## 出场逻辑变体

### X. 高估区出场

```
IF CBBI > exit_cb OR AHR999 > exit_ahr:
    出场
```

参数空间：
| exit_cb | exit_ahr |
|:---:|:---:|
| 0.75 | 1.0 |
| 0.75 | 1.2 |
| 0.80 | 1.0 |
| 0.80 | 1.2 |

→ 4 种组合

### Y. 动量反转出场

```
IF CBBI_N日动量 < -threshold:
    出场
```

参数空间：
| N | threshold |
|:---:|:---:|
| 3 | 0.03 |
| 3 | 0.05 |
| 5 | 0.03 |
| 5 | 0.05 |

→ 4 种组合

### Z. 趋势线出场

```
IF close < SMA200 OR EMA50 < EMA200:
    出场
```

→ 1 种（固定逻辑，无参数）

## 分层筛选流程

### 第 1 层：粗筛（训练集 2022-2025）

**组合矩阵**：A×X, A×Y, A×Z, B×X, B×Y, B×Z = 6 类策略

每类取 2-3 组代表性参数 → **约 15-18 组回测**

**筛选标准**：
1. 总收益 > +88%（超过 BuyAndHold）
2. 交易次数 >= 5（统计显著性）
3. 最大回撤 < -30%

**输出**：Top 3 策略类型 + 参数组合

### 第 2 层：精筛（滚动窗口验证）

对 Top3 各展开 5-8 组参数变体，用 `val_rolling.py` 的 7 段滚动窗口验证。

**筛选标准**：
- 7 段均值 > +25%（接近 BAH 的 +27%）
- 任何单段回撤 < -25%
- 无连续 2 段亏损

**输出**：最终策略 + 最优参数

### 第 3 层：验证

对最终策略做完整验证：
- `run.py` 训练集回测
- `val_rolling.py` 7 段滚动验证
- `val.py` 2026 Q1 OOS 验证
