## 2026 Q1 验证核心发现

```
BTC $94k → $76k, -16% 缓慢阴跌

4/6 策略正确空仓 (0% 损失):
  SmartHold   — EMA 死叉在年初已形成 → 不进场
  CbbiLead    — CBBI 不在恐惧区 → 不进场  
  EmaValuation — EMA 死叉 → 不进场
  Bear01      — SMA200 下方+宏观恶化 → 不进场

1/6 仍持仓亏钱:
  Cycle01     — AHR999/CBBI 不识别阴跌 (-16.5%)
```

关键结论:
1. **入场过滤比出场过滤更可靠** — 不进场远比找准时机出场容易
2. **EMA 交叉 + CBBI + SMA200 三套过滤互补** — 覆盖了不同的熊市场景  
3. **周期指标(AHR999/CBBI)不适合中期趋势** — 它们识别的是大周期顶部/底部，不是16%的阴跌

## 指标速查

| 指标 | 用途 | 获取方式 | 预热需求 |
|------|------|---------|---------|
| EMA100 | 中长趋势 | `@informative("1d")` + `ta.EMA(df, 100)` | 100天 |
| EMA200 | 长趋势 | `@informative("1d")` + `ta.EMA(df, 200)` | 200天 |
| SMA200 | 简单趋势（熊市过滤最可靠） | `@informative("1d")` + `ta.SMA(df, 200)` | 200天 |
| AHR999 | 链上估值（<0.45低估 >1.5过热） | `merge_ahr999(df, meta)` | 200天 |
| CBBI | 市场情绪（<0.3恐惧 >0.7贪婪） | `merge_cbbi(df, meta)` | 无 |
| 资金费率 | 多空失衡（<-0.05踩踏） | `merge_external_factors(df, meta, columns=[...])` | 无 |
| 稳定币增速 | 资金流向（<0外流） | `merge_external_factors(df, meta, columns=[...])` | 无 |
